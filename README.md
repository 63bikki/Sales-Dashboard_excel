# Retail Sales Dashboard in Excel

Created an interactive sales dashboard using Excel Pivot Tables, Pivot Charts, slicers, KPI cards, and correlation analysis.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Key Insights
- Analysed total sales, total orders, quantity sold, and average order value.
- Identified top-performing products and customer purchasing patterns.
